# Penguin3d - Wall tech demo
### Creates a fixed sprite image of a wall, made with custom Penguin3d engine based on OpenGL
Notice: This version was built in alpha, and was built directly into the engine code. On official releases, this will be managed by an API and seperate folder

## 🏗️ To Build
Open up the .sln file in Visual Studio and run
